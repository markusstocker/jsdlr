A JavaScript ALC Tableau Reasoner (pre-alpha)

Please note:
  • Expressions are required to have pairs for sub-expressions enclosed by parenthesis. Hence, ((A ⊓ B) ⊓ C) is OK, but (A ⊓ B ⊓ C) will be interpreted as invalid.
  • You can type concept names and parenthesis in the input field and either append it to the expression or select a radio button to append an operator.
  • The reasoning task supported is concept satisfiability (for now).
  • The input field may be used to build expressions. Reasoning tasks are performed on the bold expression displayed below the input field. Click on Reset to provide a new expression.
  • Tested and implemented with Firefox 2.0.0.3. Some things probably won't work in IE or other browsers!
